# 3D Parts

This directory contains all the necessary files for 3D printing the microscope parts. Our assembly is based on the **OpenFlexure Delta Stage** design.

## About the "modified" files

When reviewing the file list, you will notice that some files end with the word **"modified"**. 

These are **custom modifications** we had to make to the original design. They were created to correct certain geometric design errors or to update parts that presented compatibility issues in the original files. 

> **Note:** We highly recommend using these "modified" versions for printing to ensure the mechanism works correctly without the issues identified in the base version.

## Assembly Instructions and Base Files

The original files this project is based on, as well as the detailed step-by-step guide to assemble the microscope, can be found in the official OpenFlexure documentation. 

You can check the assembly steps and view the original files here:
**[OpenFlexure Delta Stage v1.2.2](https://build.openflexure.org/openflexure-delta-stage/v1.2.2/)**

Please follow the instructions in that link for the physical assembly, simply replacing the original parts with our "modified" versions where applicable.
